## <a id='environment-variable'></a>Environment Variables ##

Format of credentials in VCAP_SERVICES environment variable.

~~~xml
{
  sendgrid-dev-n/a: [
    {
      name: "sendgrid-dev-75efc",
      label: "sendgrid-dev-n/a",
      plan: "free",
      credentials: {
        username: "QvsXMbJ2rK",
        password: "HCDVOYluTv"
      }
    }
  ]
}
~~~
